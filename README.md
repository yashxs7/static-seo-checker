# Static SEO Checker

A Python Flask web application that analyzes static websites and generates a basic SEO audit report using BeautifulSoup.

## Features

- URL-based SEO audit
- Title tag analysis
- Meta description analysis
- Heading structure analysis
- Internal and external link analysis
- Image alt text checker
- Canonical tag checker
- Robots meta tag checker
- Open Graph tag checker
- Structured data detection
- robots.txt checker
- sitemap.xml checker
- SEO score out of 100
- Improvement recommendations

## Tech Stack

- Python
- Flask
- BeautifulSoup
- Requests
- HTML
- CSS

## How to Run Locally

```bash
pip install -r requirements.txt
python app.py
